Creates the cuFFT plans and allocates the required memory space.

The function starts by initializing pidx_i and pidx_j, along with input_dim, transposed_dim and output_dim. Afterwards, the cuFFT plans are created as described in Details. Finally, setWorkArea is called to allocate the device memory.

Parameters
global_sizespecifies the dimensions Nx, Ny and Nz (of the global input data)
partitionspecifies the grid size used to partition the global input data
allocatespecifies if device memory has to be allocated
