A simple reference testcase, where the complete input is gathered on a single MPI rank to compute the complete 3D FFT with cuFFT.

For a given grid , each rank starts by generating random input data for . Afterwards each rank sends the data to rank 0. Here we simply wait for all input data to arrive and compute the complete 3D FFT. Finally, rank 0 distributes the computed result to the individual ranks, such that each rank ends up with the same partition .

A simple reference testcase which measures the bandwidth in MB/s for each rank.

Each process generates a random input of size . Afterwards the same data is send to all other ranks. If cuda_aware is disabled, the input data is copied in each * iteration to pinned memory. There are multiple available options to consider:

  • opt=0: MPI uses Peer2Peer communication
  • opt=1: MPI uses All2All communication
